# Remember Me - Bible Verse Memorizing App

## Project Overview
**Remember Me** is a Bible verse memorizing application designed to help users learn and retain scripture effectively. The app provides a user-friendly interface to add, manage, and review Bible verses along with various helpful tools to enhance the memorization experience.

## Installation
To run the Remember Me app locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/remember-me-bible-app.git
   cd remember-me-bible-app
   ```

2. **Open `index.html`** in your browser:
   Just double-click the `index.html` file or open it in your favorite web browser.

## Usage
Once the app is open, you can navigate through different sections using the sidebar. The main features include:

- Adding Bible verses to your collection
- Fetching verse texts from a selected Bible version and language
- Marking verses as known or due for review
- Listening to verses read aloud

Use the buttons provided to navigate, add new verses, and manage your verse collection.

## Features
- **Manage Verses**: Add verses with references, text, versions, and languages.
- **Organization**: Separate your verses into known and due categories.
- **Audio Playback**: Listen to the verses read aloud through speech synthesis.
- **Responsive Design**: The application is built to be mobile-first, providing a good user experience on both desktop and mobile devices.

## Dependencies
The application requires the following dependencies which are included through CDN links:
- [Tailwind CSS](https://tailwindcss.com/) for styling
- [Font Awesome](https://fontawesome.com/) for icons (version 6.0.0-beta3)

No additional packages are needed for basic functionality.

## Project Structure
The project consists of the following main file:

```
/remember-me-bible-app
└── index.html         # Main HTML file that contains the app structure and logic
```

### Additional Information
- The app uses `localStorage` to store user verses, allowing for data persistence across sessions.
- JavaScript is used for managing the dynamic behaviour of the app.
